免费工具们 ¶

  • DBeaver Community ：基于JDBC的客户端，可连接市面上绝大多数关系型数据库

  • MySQL Workbench ：MySQL官方GUI管理工具

  • Oracle Instant Clients ：Oracle客户端工具，包含SQL*Plus

  • SQL Server Management Studio (SSMS) ：MSSQL官方GUI管理工具
